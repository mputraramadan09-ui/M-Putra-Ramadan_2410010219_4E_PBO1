## Tentang Program

Ini program buat laundry kiloan, jadi ceritanya ada tempat laundry terus bisa nginput pelanggan, milih jenis cucian (reguler, cuci+setrika, atau express), nanti otomatis kehitung berapa yang harus dibayar sesuai berat cuciannya. Selain itu bisa juga liat riwayat transaksi sama update status cuciannya udah selesai apa belum.

Dipilih tema ini soalnya sederhana tapi tetep bisa masukin semua konsep OOP yang diminta.

## Class

- Layanan (abstract, jadi induk)
- CuciReguler, CuciSetrika, CuciExpress (anak dari Layanan)
- Pelanggan
- Transaksi
- Main
